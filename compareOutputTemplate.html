Differences of .csv files

How to read?

1. added : shows all the new rows added to second .csv file, which were previously absent in first .csv file.

2. removed: shows all the removed rows from the first .csv file.

3. changed : shows all the changed cells of first csv file, with respect to some "key",
eg: w.r.t to key "0x580001" which column "address" selected as key, in column named "default" in first .csv file, cell value '0x0' changed to '0x1' in second .csv file.

4. columns_added : shows all the new columns added in the new version of .csv file.

5.columns_removed : shows all the removed columns from the previous .csv file.
